Порядок следования полей для импортируемых таблиц


TABLE_DT
--------
ID_DT,
OKRUG,
PREF,
DISTRICT,
YARD_OWNER_NAME,
OBJECT_ADDRESS


TABLE_ODH
---------
ID_ODH,
OKRUG,
OBJECT_NAME,
CUSTOMER_NAME,
CONTRACTOR_NAME


TABLE_CONTAINER_AREA
--------------------
ID_CONTAINER_AREA,
YARD_ID,
OBJECT_ADDRESS,
OKRUG,
DISTRICT_NAME


Требования:
Кодировка - UTF-8.
Для каждой записи в таблице, идентификатор объекта должен быть уникальным целым числом.
Разделитель полей - запятая.
Лучше при экспорте классификатора сохранить текстовые поля в кавычках -
на случай, если внутри текстового поля присутствует символ запятой, который может стать неправильным разделителем при импорте.
Заголовки полей в таблицу НЕ ВКЛЮЧАЕМ.
